## Severity Model

- `P0`: visual artifact, overlap, blank/black blob, unsafe copy, console/page error, horizontal overflow.
- `P1`: component pattern mismatch, wrong hierarchy, wrong route composition, bad density, missing first-viewport priority, control looks usable but is weak or misleading.
- `P2`: spacing, tone, casing, chip/icon/line-height mismatch, incomplete hover/focus proof, accessibility label gap.
- `P3`: polish, future backlog, documentation-only clarification, test coverage expansion.
